# Gemini 프롬프트 예시 모음

이번 수행평가에서는 학교 계정 Gemini를 사용할 수 있습니다.
단, 실제 개인정보를 입력하지 말고, 최종 코드는 본인이 이해하고 설명할 수 있어야 합니다.

---

## 1. 주제 찾기

```text
나는 고등학교 정보 수행평가로 파이썬 자유 주제 미니 프로젝트를 만들려고 해.
조건문, 반복문, 리스트, 2차원 리스트, 함수 정도를 배웠어.
내 관심사는 [게임/음식/운동/음악/영화/공부/여행/학교생활]이야.
2차원 리스트와 함수를 사용하기 좋은 프로젝트 주제 5개를 추천해줘.
각 주제마다 입력, 처리, 출력 구조도 함께 설명해줘.
```

---

## 2. 문제 분해와 모델링 도움받기

```text
내 프로젝트 주제는 [주제]야.
이 프로그램을 입력-처리-출력 구조로 정리해줘.
단, 파이썬 초보자가 구현할 수 있도록 조건문, 반복문, 2차원 리스트, 함수 수준에서만 설명해줘.
완성 코드는 쓰지 말고 설계 방향만 알려줘.
```

---

## 3. 2차원 리스트 구조 만들기

```text
내 프로젝트 주제는 [주제]야.
이 주제에 맞는 2차원 리스트 구조를 제안해줘.
한 행이 무엇을 의미하는지, 각 열이 무엇을 의미하는지 표로 설명해줘.
예시 데이터는 6개 정도만 만들어줘.
```

---

## 4. 함수 분리 도움받기

```text
내 프로젝트는 [주제]야.
입력은 [입력], 처리는 [처리], 출력은 [출력]이야.
이 프로그램을 함수 3개 정도로 나누고 싶어.
함수 이름, 역할, 매개변수, 반환값을 표로 제안해줘.
완성 코드는 쓰지 말고 함수 설계만 해줘.
```

---

## 5. 코드 오류 질문하기

```text
아래 코드는 내가 직접 작성한 파이썬 코드야.
오류가 나는 이유를 설명해줘.
단, 전체 코드를 새로 작성하지 말고,
1. 오류 원인
2. 수정해야 할 부분
3. 내가 직접 고칠 수 있는 힌트
형태로 알려줘.

[내 코드 붙여넣기]
```

---

## 6. 테스트 사례 만들기

```text
내 프로그램 주제는 [주제]야.
입력값에 따라 결과가 달라지는 프로그램이야.
정상 사례 2개와 예외 사례 1개를 만들어줘.
각 사례마다 입력값, 예상 결과, 확인해야 할 점을 표로 정리해줘.
```

---

## 7. 보고서 작성 방향 점검하기

```text
내가 만든 파이썬 프로젝트 보고서를 작성하려고 해.
주제는 [주제]이고, 2차원 리스트와 함수를 사용했어.
보고서에 꼭 들어가야 할 항목을 알려줘.
단, 보고서 문장을 대신 완성하지 말고 내가 직접 채울 수 있는 틀로 제시해줘.
```

---

## 8. AI 활용 성찰 작성 도움받기

```text
나는 Gemini의 도움을 받아 파이썬 프로젝트를 만들었어.
Gemini가 제안한 내용을 그대로 쓰지 않고 내가 수정한 점을 보고서에 쓰려고 해.
AI 활용 성찰에서 어떤 내용을 쓰면 좋을지 질문 목록으로 정리해줘.
```
